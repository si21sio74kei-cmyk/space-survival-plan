# 太空数据中心功能说明

## 📡 功能概述

深空AI生存系统现已集成**15个权威太空/地球数据API**，分为4批次实现，为系统提供实时太空环境监控和地球态势感知能力。

---

## 🚀 已集成的API列表

### 第一批：核心空间天气API（NASA DONKI）

| API名称 | 端点 | 功能 |
|---------|------|------|
| 太阳风暴(CME) | `/api/space-weather/storms` | 监测日冕物质抛射事件 |
| 太空辐射(RBE) | `/api/space-weather/radiation` | 辐射带增强事件监控 |
| 太阳耀斑(FLR) | `/api/space-weather/flares` | X/M/C级耀斑检测 |
| **综合摘要** | `/api/space-weather/summary` | 风险等级评估与警告 |

**应用场景**：
- ⚠️ 辐射预警联动应急响应
- 🔋 AI决策能源分配策略
- 🛡️ 宇航员出舱安全评估

---

### 第二批：实时态势感知API

| API名称 | 端点 | 功能 |
|---------|------|------|
| ISS位置 | `/api/iss/position` | 国际空间站实时经纬度 |
| 在轨宇航员 | `/api/astronauts` | 当前太空人员统计 |
| SpaceX发射 | `/api/spacex/latest` | 最新火箭任务信息 |
| SpaceX火箭 | `/api/spacex/rockets` | 全系列火箭参数库 |

**应用场景**：
- 🛰️ 太空任务实时监控大屏
- 👨‍🚀 载人航天状态看板
- 🚀 商业航天动态追踪

---

### 第三批：视觉与环境增强API

| API名称 | 端点 | 功能 |
|---------|------|------|
| 火星照片 | `/api/mars/photos` | Curiosity/Perseverance高清影像 |
| 地球照片 | `/api/earth/photos` | EPIC卫星全彩地球照 |
| 月球数据 | `/api/moon` | 天体物理参数库 |
| 行星数据 | `/api/planets` | 八大行星完整参数 |

**应用场景**：
- 🌍 深空控制中心背景系统
- 🖼️ 天文壁纸自动更新
- 📊 多星球运行轨迹模拟

---

### 第四批：地球环境监控API

| API名称 | 端点 | 功能 |
|---------|------|------|
| 自然灾害 | `/api/earth/disasters` | 火山/火灾/风暴事件 |
| 全球地震 | `/api/earthquakes` | 5级以上地震实时数据 |
| 空气质量 | `/api/air-quality` | AQI及污染物浓度 |
| 澳门天气 | `/api/weather/macau` | 中文天气+24h预报 |

**应用场景**：
- 🌪️ 全球灾害监控大屏
- 📈 GIS地理信息防灾平台
- 🏙️ 智慧城市外部环境模拟

## 🎨 前端界面

### 访问方式

1. 启动应用后，点击左侧导航栏的 **"太空数据中心"** 图标（卫星图标）
2. 页面将自动加载4个面板的数据：
   - 🌞 **太空天气 SPACE WEATHER**
   - 🛰️ **实时态势 SITUATIONAL AWARENESS**
   - 🖼️ **深空影像 DEEP SPACE IMAGERY**
   - 🌍 **地球监控 EARTH MONITORING**

### 显示内容

#### 太空天气面板
- 风险等级（低/中/高/严重）
- 警告信息列表
- 事件统计（太阳风暴/辐射/耀斑数量）

#### 实时态势面板
- ISS当前位置（纬度/经度/时间）
- 在轨宇航员名单及所属飞船
- SpaceX最新发射任务详情

#### 深空影像面板
- 火星地表照片网格（6张缩略图）
- 月球核心参数（质量/半径/密度/重力）

#### 地球监控面板
- 近期自然灾害列表（前5条）
- 最近地震记录（震级/地点/时间）
- 空气质量指数（AQI颜色编码）

## 🔧 技术实现

### 后端架构

```
backend/space_data_api.py
├── SpaceDataAPI类
│   ├── 缓存机制（避免频繁请求）
│   ├── 错误处理（优雅降级）
│   └── 15个API封装方法
├── 综合数据接口
│   ├── get_all_space_weather()
│   ├── get_situational_awareness()
│   ├── get_visual_enhancement_data()
│   └── get_earth_environment_data()
```

### 缓存策略

| 数据类型 | 缓存时长 | 原因 |
|---------|---------|------|
| ISS位置 | 10秒 | 高速移动，需实时更新 |
| 太空天气 | 1小时 | 变化较慢 |
| 火星照片 | 24小时 | 静态历史数据 |
| 行星参数 | 24小时 | 几乎不变 |
| 地震/灾害 | 30分钟 | 中等频率更新 |

### 前端集成

```javascript
// app.js
async function loadSpaceDataModule() {
    // 并行加载4组数据
    await Promise.all([
        fetch('/api/space-weather/summary'),
        fetch('/api/situational-awareness'),
        fetch('/api/visual-enhancement'),
        fetch('/api/earth-environment')
    ]);
}
```

## ⚙️ 配置说明

### NASA API密钥

默认使用 `DEMO_KEY`（有速率限制）。如需更高配额：

1. 访问 https://api.nasa.gov/
2. 注册获取个人API Key
3. 修改 `backend/space_data_api.py` 第11行：
   ```python
   NASA_API_KEY = "YOUR_API_KEY"
   ```

### 其他API密钥

- **WAQI空气质量**: 使用 `demo` token（有限制），注册地址 https://aqicn.org/api/
- **腾讯天气**: 无需密钥，直接调用

## 🐛 常见问题

### Q1: API返回"加载失败"

**原因**：网络连接问题或API限流

**解决**：
- 检查网络连接
- 等待几分钟后重试（DEMO_KEY有速率限制）
- 申请个人API Key提高配额

### Q2: 火星照片不显示

**原因**：NASA图片服务器可能暂时不可用

**解决**：
- 刷新页面重试
- 尝试不同的sol（火星日）参数

### Q3: 数据更新不及时

**原因**：缓存机制导致

**解决**：
- 缓存是故意设计的，避免频繁请求
- 如需强制刷新，重启Flask应用
