## 🚀 系统概述

**深空AI生存系统**是一个模拟太空任务中生存资源管理的智能系统。它通过AI算法实时监控和预测：

- **食物稳定性** - 食物库存和营养状况
- **医疗安全性** - 医疗物资和冷链管理
- **能源持续性** - 能源分配和使用效率
- **氧气安全** - 舱内空气质量和环境控制
- **水资源** - 饮用水储备和循环利用

### 核心特性

✅ **智能预测** - 基于当前状态自动计算预计生存天数和未来趋势  
✅ **多系统联动** - 各模块相互影响，真实模拟资源消耗  
✅ **实时反馈** - 每次操作后立即看到预测结果  
✅ **AI决策支持** - 提供优化建议和风险预警

#### 3️⃣ AI引擎处理阶段

后端`ai_engine.py`接收请求并处理：

**关键改进（已修复）：**
- ✅ 更新状态数据（如食物库存）
- ✅ 重新计算相关指标（如食物稳定性）
- ✅ 调用 `get_current_status()` 触发智能预测

#### 4️⃣ 智能预测计算阶段

`get_current_status()` 函数执行以下计算：

**A. 预计生存天数计算**
```python
# 基于资源衰减率的智能计算
crew_count = max(1, status['crew_count'])
food_days = status['food_stability'] / (2.0 * crew_count / 4.0)
energy_days = status['energy_level'] / 1.0
oxygen_days = status['oxygen_level'] / 0.5
water_days = status['water_reserve'] / (1.5 * crew_count / 4.0)

# 取最小值作为瓶颈资源
estimated_survival_days = min(food_days, energy_days, oxygen_days, water_days)
```

**B. 预测时间线生成**
```python
# 推演未来30/60/90/120天的状态
daily_decay = {
    'food': 2.0 * crew_count / 4.0,      # 每日食物消耗
    'energy': 1.0,                        # 每日能源消耗
    'oxygen': 0.5,                        # 每日氧气消耗
    'water': 1.5 * crew_count / 4.0,      # 每日水消耗
    'medical': 0.3                        # 每日医疗物资消耗
}

predictions = []
for day in [30, 60, 90, 120]:
    # 计算未来各资源剩余量
    food_future = max(0, food_stability - daily_decay['food'] * day)
    energy_future = max(0, energy_level - daily_decay['energy'] * day)
    # ... 其他资源
    
    # 计算未来的生存指数
    predicted_index = (
        food_future * 0.2 +
        medical_future * 0.3 +
        energy_future * 0.2 +
        oxygen_future * 0.2 +
        water_future * 0.1
    )
    predictions.append(predicted_index)
```

## 🧠 智能预测机制

### 预测算法详解

#### 1. 生存指数计算公式

```python
survival_index = (
    food_stability × 0.2 +      # 食物权重20%
    medical_safety × 0.3 +      # 医疗权重30% ← 最重要！
    energy_level × 0.2 +        # 能源权重20%
    oxygen_level × 0.2 +        # 氧气权重20%
    water_reserve × 0.1         # 水资源权重10%
)
```

**为什么医疗权重最高？**
- 在太空环境中，医疗问题可能是致命的
- 缺乏药物可能导致小病变成大病
- 冷链失效会导致疫苗和血液制品失效

#### 2. 预计生存天数计算

```python
# 计算每种资源的可用天数
food_days = food_stability / (2.0 × crew_count / 4.0)
energy_days = energy_level / 1.0
oxygen_days = oxygen_level / 0.5
water_days = water_reserve / (1.5 × crew_count / 4.0)

# 瓶颈资源决定生存时间
estimated_survival_days = min(food_days, energy_days, oxygen_days, water_days)
```

**关键洞察：**
- 生存时间由**最短缺的资源**决定
- 即使其他资源充足，只要一种耗尽就结束
- 这就是为什么要平衡发展各系统

#### 3. 预测时间线生成

```python
# 对未来4个时间点进行推演
for day in [30, 60, 90, 120]:
    # 计算未来各资源剩余量
    food_future = max(0, food_stability - daily_decay['food'] × day)
    # ... 其他资源
    
    # 计算未来的生存指数
    predicted_index = (
        food_future × 0.2 +
        medical_future × 0.3 +
        energy_future × 0.2 +
        oxygen_future × 0.2 +
        water_future × 0.1
    )
    predictions.append(predicted_index)
```

**预测的意义：**
- D+30：短期趋势（是否有立即危险）
- D+60：中期规划（是否需要补给）
- D+90：长期战略（是否需要调整策略）
- D+120：远景预测（任务可行性评估）

```
- 医疗：30%
- 食物：20%
- 环境：35%
- 其他：15%
```

### 4. 环境控制策略

#### 🌡️ 目标设定
- **氧气**：维持在20-22%（过低危险，过高浪费）
- **温度**：20-24°C（舒适区，节省能源）
- **湿度**：45-55%（防止设备腐蚀和细菌滋生）

#### ⚠️ 警报阈值
- CO₂上限：0.5%（超过需紧急通风）
- 氧气下限：18%（低于需增氧）
- 温度异常：±5°C（超出需调节）

### 5. 预测分析方法

#### 📈 解读预测时间线

**健康曲线（缓慢下降）：**
```
D+0:  85%
D+30: 75%  ← 每月降10%，可控
D+60: 65%
D+90: 55%
D+120: 45%
结论：资源充足，任务可行
```

**危险曲线（快速下降）：**
```
D+0:  60%
D+30: 35%  ← 每月降25%，危险！
D+60: 10%  ← 即将耗尽
D+90: 0%
D+120: 0%
结论：需要立即补充资源或减少消耗
```

**平稳曲线（几乎不降）：**
```
D+0:  90%
D+30: 88%
D+60: 86%
D+90: 84%
D+120: 82%
结论：资源过剩，可以支援其他任务
```

## ❓ 常见问题

### Q1: 为什么我添加了食物，但预计生存天数没变？

**A:** 可能的原因：
1. **食物数量太少**：添加的量不足以显著提升食物稳定性
2. **其他资源是瓶颈**：即使食物充足，如果氧气或能源不足，生存天数仍由它们决定
3. **乘员太多**：消耗率太高，新增食物很快被消耗

**解决方法：**
- 查看雷达图，确认哪项是最低的资源
- 优先补充瓶颈资源
- 考虑减少乘员数量或降低消耗率

---

### Q2: 为什么预测时间线是直线下降的？

**A:** 这是正常的！预测基于**线性衰减模型**：
- 每天消耗固定量的资源
- 所以生存指数随时间线性下降
- 如果看到曲线突然下降，说明某项资源即将耗尽

**如何改善：**
- 定期补充资源（添加食物、医疗物品等）
- 降低消耗率（启用节能模式、紧急配给等）
- 优化能源分配，提高效率

---

### Q3: 生存指数和预计生存天数有什么区别？

**A:** 
- **生存指数**（0-100%）：当前系统的综合健康度
  - 反映"现在有多好"
  - 由五项指标的加权平均计算
  
- **预计生存天数**（天数）：还能活多久
  - 反映"还能撑多久"
  - 由最短缺的资源决定

**举例：**
```
情况A：
- 生存指数：85%（很好）
- 预计生存：10天（很短）
原因：虽然各项指标都不错，但总量少，很快耗尽

情况B：
- 生存指数：60%（一般）
- 预计生存：100天（很长）
原因：虽然当前状态一般，但总量大，能撑很久

结论：两个指标都要看！
```
