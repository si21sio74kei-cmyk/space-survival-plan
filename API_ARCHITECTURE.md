## 🔄 数据流向

### 1. 太空数据采集流程
```
外部API → SpaceDataAPI类 → 缓存层 → Flask路由 → 前端展示
   │           │              │          │          │
   │           ├─ NASA        │          │          │
   │           ├─ SpaceX      │          │          │
   │           ├─ Open-Notify │          │          │
   │           └─ 其他        │          │          │
   │                          │          │          │
   └─ 15个独立方法            │          │          │
                              ├─ 智能缓存 │          │
                              ├─ 错误处理 │          │
                              └─ 降级策略 │          │
                                         │          │
                                         ├─ 21个端点 │
                                         │          │
                                         └─ JSON响应
```

### 2. AI决策流程
```
系统状态 → ai_engine.py → GLM-4.5-Air → 决策结果 → 执行动作
   │            │              │            │          │
   ├─ 生存指数  ├─ 状态收集    ├─ API调用   ├─ 分析报告 │
   ├─ 资源水平  ├─ 上下文构建  ├─ Token消耗 ├─ 预警信息 │
   ├─ 环境参数  ├─ Prompt工程  ├─ 速率限制  ├─ 优化建议 │
   └─ 历史数据  └─ 逻辑判断    └─ 错误处理  └─ 自动执行
```

### 3. 用户操作流程
```
前端界面 → HTTP请求 → Flask路由 → 业务逻辑 → 数据返回
   │           │           │           │          │
   ├─ 点击按钮 ├─ GET/POST ├─ 参数验证 ├─ 数据库   ├─ JSON
   ├─ 表单提交 ├─ 认证检查 ├─ 权限验证 ├─ API调用  ├─ 状态码
   ├─ 实时刷新 ├─ 速率限制 ├─ 输入清洗 ├─ AI决策   ├─ 错误信息
   └─ 数据展示 └─ CORS处理 └─ 日志记录 └─ 缓存更新 └─ 成功响应
```

## 🎯 关键路径

### 核心功能依赖链
```
生存监控 → 太空天气API → 风险评估 → AI决策 → 紧急协议
    ↓
资源管理 → 食物/医疗API → 库存预警 → 配给调整
    ↓
环境控制 → 地球环境API → 参数优化 → 自动调节
    ↓
态势感知 → ISS/宇航员API → 实时更新 → 可视化展示
```

### 故障隔离
```
外部API故障 → 缓存降级 → 备用数据 → 友好提示
    ↓
AI API故障 → 规则引擎 → 默认策略 → 人工介入
    ↓
数据库故障 → 内存模式 → 临时存储 → 数据恢复
```
