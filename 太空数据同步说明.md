# 太空数据同步到总控制台 - 功能说明

## 🎯 功能概述

太空数据中心获取的**实时NASA太空天气数据**现已自动同步到总控制台，实现全局态势感知。

---

## 📊 同步机制

### 数据流向

```
NASA DONKI API (真实数据)
    ↓
space_data_api.py (缓存处理)
    ↓
ai_engine.py::simulate_step() (每次模拟步骤调用)
    ↓
辐射等级调整 + AI决策联动
    ↓
前端总控制台显示 (顶部状态栏指示器)
```

### 同步频率

- **API调用**: 每小时1次（缓存机制）
- **状态更新**: 每5分钟1次（前端定时刷新）
- **风险响应**: 实时（检测到高风险立即触发应急协议）

## ❓ 常见问题

### Q1: 总控制台没有显示太空天气指示器？

**检查**：
1. 确认SPACE_DATA_ENABLED为True（查看控制台输出）
2. 刷新页面（Ctrl+F5强制刷新）
3. 检查浏览器控制台是否有JavaScript错误

### Q2: 指示器一直显示"稳定"，没有变化？

**原因**：NASA API返回的数据确实是低风险状态

**解决**：
- 这是正常现象，太阳活动本身就有周期性
- 可以等待几小时后再观察
- 或临时修改API返回数据测试

### Q3: API调用失败怎么办？

**自动处理**：
- 系统会自动降级为随机模拟模式
- 总控制台仍会显示，但标注"（模拟数据）"
- 不影响其他功能正常运行
